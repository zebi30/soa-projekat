Tour lifecycle test

Draft creation, publish validation, gRPC-backed publish/list, and archive/activate checks.

Flow

  1. Create or log in as a guide and tourist.
  2. Create a guide-owned draft tour with at least one tag.
  3. Add two key points so lengthKm is calculated.
  4. Add at least one transport time.
  5. Publish the draft through POST /api/tours/:id/publish.
  6. List published tours as a tourist through GET /api/tours/published.
  7. Archive, reactivate, and re-check visibility.
